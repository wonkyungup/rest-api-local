# The express + maria  Rest Api Test

## Environment Introduction
- node-express (https://expressjs.com)
- mariadb (https://mariadb.com)

## How to use
1. install node_module
```
npm install
```

2. run express
```
npm run start
```

## Url

    GET /v1/user
    DELETE /v1/user
    POST /v1/user/register
    POST /v1/auth
    GET /v1/auth
